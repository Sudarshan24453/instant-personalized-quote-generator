# Instant Personalized Quote Generator
Instant personalized insurance quote generator using Machine Learning (Linear Regression) built in Jupyter Notebook.

This demonstrates an instant personalized insurance quote generator using Machine Learning.

## Description
A Linear Regression model is trained on a sample dataset to predict insurance quote amounts based on user inputs such as age and income.

## Features
- Machine Learning based quote prediction
- Instant output using trained model
- Implemented fully in Jupyter Notebook

## Model Used
- Linear Regression

## How to Run
1. Open the notebook `instant_quote_generator.ipynb`
2. Run all cells sequentially
3. Use the `generate_quote(age, income)` function to get instant quotes
